## üìö Reference Sources

As a reference for most achievement texts, the WotLK achievements from Wowhead can be used:

https://www.wowhead.com/wotlk/achievements

Only for the Quest Storylines in Outland, the WoW Retail achievement texts were used.

Some achievements include the quest ID or NPC ID in their variable name.  
You can use these IDs to find the correct reference.

### Example
QUEST_835 = "Securing the Lines"

This refers to:

https://www.wowhead.com/tbc/quest=835/securing-the-lines

Using these links helps ensure accurate translations.

## ‚ö†Ô∏è Please Do Not Use AI

Please do **not** use AI tools for translation.

I tested this myself with the German version, and the results were unreliable:
- Original texts were not preserved
- Content was hallucinated
- Terminology was inconsistent

Some achievements (especially in the PvP category) are custom-made and not originally from Blizzard, so manual and creative translation is required.

### Preferred Method: GitHub

If you are familiar with GitHub, this is the best way to contribute.

Repository:
https://github.com/fgaensslen/AnniversaryAchievements

Steps:

1. Fork or clone the repository
2. Create a new branch
3. Add your translated file
4. Open a pull request
